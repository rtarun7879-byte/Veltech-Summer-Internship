About CropSense

🌾 What is CropSense?

CropSense is an AI-powered crop recommendation system built during a machine learning internship. It uses a Random Forest Classifier trained on soil nutrient and climate data to suggest the most suitable crop for a given set of conditions.

🧠 The ML Model

The model is a Random Forest Classifier with 100 decision trees, trained on the Crop Recommendation Dataset containing 2,200 samples across 22 crop types. It achieves high accuracy by combining multiple decision trees and voting on the outcome.

📥 Input Features

  • N — Nitrogen content in soil (kg/ha)
  • P — Phosphorus content in soil (kg/ha)
  • K — Potassium content in soil (kg/ha)
  • Temperature — Average temperature (°C)
  • Humidity — Relative humidity (%)
  • pH — Soil acidity/alkalinity
  • Rainfall — Annual rainfall (mm)
